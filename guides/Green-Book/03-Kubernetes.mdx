# 3.0 Kubernetes

Kubernetes is open-source platform for managing containerized services. Kubernetes offers many services for service automation, monitoring and orchestration. Kubernetes makes easier to manage scalable microservices, which means that resources are better used where they are needed and when they are needed. Kubernetes is a tool that knows what containers each application needs.

Following video explains Kubernetes basic idea in a clear and fun way.
